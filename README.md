# VisionSoC 🖼️

AI-powered image enhancement using ESRGAN (Enhanced Super-Resolution GAN).

## Features
- 4× image upscaling
- Flask web interface
- PyTorch + OpenCV backend

## Setup
pip install flask torch opencv-python numpy

## Run
python test.py

## Usage
1. Open http://127.0.0.1:5000
2. Upload a low-res image
3. Click Enhance
4. Download the result
